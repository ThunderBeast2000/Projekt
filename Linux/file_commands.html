Fileokkal kapcsolatos parancsok



cat:Ezt a parancsot több célra is használhatjuk, mint pl: fileok létrehozása, kiiratása, összefűzése.

Hogyha filet akarunk létrehozni, akkor kilépni a CTRL+D billentyűkombináció lenyomásával tudunk.


sort:Egy file tartalmának a rendezése. Hogyha alfanumerikusan akarjuk rendezni, akkor a -f kapcsolót kell használni. Megfordítani a -r kapcsolóval tudjuk.


cp (copy):Egy file vagy directory másolása. A -i kapcsolóval kaphatunk meg több információt a másolásról. -r/-R kapcsolókkal rekurzívan másolhatjuk át a fileokat egy directoryból. A -f kapcsolóval "erővel" tudjuk átmásolni a fileokat. Kétfajta másolás van, az egyik az úgynevezett softlink, amit a -s kapcsolóval tudunk használni. A másik a hardlink -l kapcsolóval használhatjuk.


rm (remove):Egy file vagy directory végleges törlése, azaz miután kiadjuk a parancsot nem tudjuk visszaállítani. Itt is használhatjuk a copynál felsorolt első három kapcsolón ezen kívül a -v kapcsoló (verbous) több információt ad, és a -d kapcsolót, amivel nem töröljük a tartalmát.


mv (move):Egy file vagy directory áthelyezése egyik helyről a másikra. Hogyha egy directoryt vagy filet akarsz átnevezni akkor lehet arra is használni. Itt is használhatjuk a megszokott három kapcsolót. (-r -f -i)


Parancsok , vonatkozóan.
